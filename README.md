## 📝 关于本项目

这是一个**纯记录向**的网页项目，最初是作者自用的个人工具，投入了大量时间和精力去设计、迭代和优化。后来决定开源分享，希望能帮助到有同样需求的朋友。

### 💡 项目定位

严格来说，这**不是一个产品**，而是一个**个人记录模板**。虽然为了公开做了一些功能扩展，但核心用户仍是作者本人。如果你认可这个项目并想要更多自定义功能，完全可以：

1. Clone 代码到本地
2. 使用 AI 辅助编程工具（如 Cursor、GitHub Copilot）做微调
3. 只需要一点点前端基础，就能拥有独属于自己的记录网页

### 🔒 数据安全说明

这是一个**纯前端网页**，所有数据存储在浏览器本地：

- **文本数据**：存储在 LocalStorage
- **图片数据**：存储在 IndexedDB

查看方式（以 Chrome 为例）：
1. 按 F12 或右键 → 检查，打开开发者工具
2. 点击 Application 选项卡
3. 在 Local Storage 下查看文本数据
4. 在 IndexedDB 下查看图片数据

如有数据安全顾虑，欢迎 clone 代码查看具体逻辑。

### 🖥️ 显示兼容性说明

如果发现样式存在明显的膨胀或压缩，可以尝试调整浏览器缩放比例。

- **开发环境**：屏幕分辨率 2560×1600，缩放 150%
- **已适配**：2560×1600、1920×1200 分辨率下的常见缩放比例
- **查看配置**：Windows 设置 → 系统 → 屏幕

## ✨ 核心功能

- **💰 记账管理** - 记录收支、分类管理、数据分析
- **😴 睡眠记录** - 追踪睡眠时长、质量评估、趋势分析
- **📚 学习记录** - 学习时长统计、分类管理、进度追踪
- **📅 日常记录** - 微信步数、三餐打卡、习惯统计
- **📝 日记速记** - 日记、歌词、摘抄快速记录
- **🎵 音乐记录** - 乐记、歌词收藏
- **📖 阅读记录** - 书记、摘抄管理
- **🏥 病记管理** - 健康记录追踪

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

应用将在 `http://localhost:5173` 启动。

## 💡 首次使用

首次访问时，系统会询问是否导入示例数据。示例数据包含：
