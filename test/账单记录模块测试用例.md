# 账单记录模块测试用例

## 测试概述
本文档包含账单记录模块的完整测试用例，涵盖所有功能点和边界条件测试。

## 一、记录添加功能测试

### 1.1 支出记录添加

#### TC-001: 正常添加支出记录
**前置条件：** 已进入账单记录页面  
**测试步骤：**
1. 选择"支出"模式
2. 选择日期（如：2025-01-15）
3. 输入金额：100.50
4. 选择分类：餐饮
5. 输入备注：午餐
6. 点击"添加记录"

**预期结果：**
- 提示"添加成功"
- 记录出现在列表中
- 表单金额和备注字段清空，日期和分类保持不变

#### TC-002: 收入记录添加
**前置条件：** 已进入账单记录页面  
**测试步骤：**
1. 选择"收入"模式
2. 选择日期（如：2025-01-15）
3. 输入金额：5000
4. 选择分类：工资收入
5. 输入备注：1月工资
6. 点击"添加记录"

**预期结果：**
- 提示"添加成功"
- 记录出现在收入列表中
- 表单金额和备注字段清空

#### TC-003: 添加无备注记录
